About Me

 

Well, hello there! My name is is Lautaro Emanuel and I’m a software engineering student at San Luis National University in San Luis, Argentina.

You can contact me by sending an email to emlautarom1@gmail.com. I’m also on Twitter and if you’re interested in my projects you can check my GitHub account.

Some projects I’ve worked on:

  • Proyecto Paraísos: Mobile/Web PWA for tracking sick trees in my hometown, aggregate collected data and present it in a readable format to the community.

  • Chip 8 Emulator: Chip 8 Virtual Machine implemented in Rust.

  • Information Theory: Cross-platform Desktop App for data Protection and Compression using Hamming’s and Huffman’s algorithms.

  • W-Pong: Web based Pong built with JavaScript, CSS and HTML.

  • Notes FX: JavaFX / SQLite application to store, view, edit and delete personal notes.
